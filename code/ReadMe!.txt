To run the code:


1. Drag the video you want to use for analysis into the code folder
2. Open the code folder in Visual Studio Code
3. Open the sketch.js file
4. Change the video name to the video you wanted to use
5. Check if you have the local server extention, download it if you don't have it installed
6. Run the script by clicking on the go live button, the script will run on a local port
7. You will have an automatic csv file downloaded at the end of the video for further analysis
